================================================================================
  API ENDPOINT
================================================================================

Endpoint: POST /api/ranking/

Example Request:
{
  "hitter_metric": "ops",
  "pitcher_metric": "era",
  "season": 2025,
  "details": false
}

Example Response:
{
  "AL": [
    ["New York Yankees", 2.145],
    ["Houston Astros", 1.892]
  ],
  "NL": [
    ["Los Angeles Dodgers", 2.567]
  ]
}

================================================================================
  AVAILABLE METRICS
================================================================================

Hitting Metrics (Higher is Better):
  avg, ops, hr, rbi, r, h, obp, slg

Pitching Metrics:
  era (lower is better)
  whip (lower is better)
  so, w, l (lower is better), bb (lower is better)
